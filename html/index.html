Welcome to FlyLight

Here you can configure and monitor FlyLight.
Please note that after each configuration change, FlyLight will restart, losing its current history.

About FlyLight

FlyLight is software for reading from weather sensors and changing the colour of a lamp depending on if it is flyable or not. All the parameters can be controlled by you.
